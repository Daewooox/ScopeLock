# Реестр уроков (lessons registry)

Дедуплицированный реестр сигнатур уроков из рефлексий проекта. Пока пуст — наполняется автоматически на этапе `/reflect` (шаг 3a) после каждой задачи.

## Зачем этот файл

Единственный документ, по которому `/reflect` сверяет «а ловили ли мы такой урок раньше».
Решает проблему «урок мелькнул один раз, замолчал на много задач, вынырнул снова» — без перечитывания всех рефлексий: новый урок сверяется с ЭТИМ файлом (одно чтение), а не со всем корпусом рефлексий.

## Формат строки

`- <правило одной строкой> — #NNNN, #MMMM [статус]`

- `[✓ <откуда>]` — урок уже продвинут в always-loaded память проекта (auto-memory / правила в CLAUDE.md|AGENTS.md). В реестре это **указатель, не дубль** — не копируй сюда тело правила.
- `[★N]` — повторяющийся урок (встретился N раз), ещё не продвинут → кандидат на продвижение.
- `[×N]` / `[×1]` — кандидат, встретился N раз / один раз, ждёт повторения.

## Что делает `/reflect` с этим файлом (инкрементально)

1. Сформулировал урок задачи → сверил с сигнатурами в нужной категории.
2. Есть совпадение → дописал `#задачу`, поднял счётчик.
3. Нет → добавил строку `[×1]`.
4. **Строгий порог продвижения:** урок попадает в always-loaded память только если он `×2+` **И** глобально-применим как правило поведения. Большинство `★`/`×` остаётся жить здесь — это держит always-loaded память худой.
5. Изредка — уборка: слить разъехавшиеся дубли, продвинутые свернуть в указатель.
