## Типичные файлы (как ориентир)

- `foundations.md` — цвета, типографика, spacing, border-radius, тени, design tokens
- `components-basic.md` — базовые компоненты (Button, Input, Modal, Card, ...)
- `states-icons-animations.md` — hover/focus/disabled, иконки, transitions
- `layout-responsive-a11y.md` — page layout, breakpoints, доступность

Заводи файлы по мере того, как они нужны. Не плоди пустых заголовков.

## Когда заводить новый файл

- Появилась устойчивая тема (≥3 связанных решения в проекте).
- В `/reflect` или `/creative` всплыл вопрос, который повторится — стоит зафиксировать.
