# @scopelock/core

Deterministic contracts, drift checks, scheduling, hook policy, worktree
isolation, and release-evidence schemas for
[ScopeLock](https://github.com/Daewooox/ScopeLock).

Most users should install `@scopelock/cli`. This package is the reusable
library surface for integrations.

MIT licensed. Requires Node.js 22 or newer.
